# Homelab

自宅でProxmoxを使って、仮想化環境・Minecraftサーバー・ネットワーク環境を構築しています。

## 使用技術

- Proxmox

- Linux

- Docker

- Minecraft Server

- Omada

- Cloudflare Tunnel

- VLAN

## 目的

インフラ、ネットワーク、セキュリティの学習と、将来的な副業・起業のため。

## 構成

### Node 1

- CPU: i7-3632QM

- Memory: 16GB

- Storage: SSD

### Node 2

- CPU: i5-12400F

- Memory: 32GB

- GPU: RTX3060 12GB

- Storage: SSD + HDD

### Network

- Router: Omada ER707-M2

- Switch: 2.5GbE Switch

- VLAN: 管理用 / クライアント用 / サーバー用
